# Employee Salary Manager

A simple employee salary management system developed in C#.

## 📌 About the Project

This project was created to practice object-oriented programming concepts, logic development, and C# fundamentals.

The system allows employee data management and salary calculations through a console application.

## 🚀 Features

- Register employees
- Store employee information
- Calculate salaries
- Display employee data
- Organized console interface

## 🛠️ Built With

- C#
- .NET
- Object-Oriented Programming (OOP)

## 📂 Project Structure

- `Program.cs` → Main application flow  
- `AccountData.cs` → Employee data structure  
- `.sln` and `.csproj` → Project configuration files
